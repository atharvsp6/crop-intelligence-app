├── Title (24pt Montserrat Bold, Green #2E7D32)

## 🎨 **VISUAL DESIGN ELEMENTS**

### **Icon Library:**
```
Technology Icons:
⚛️ React | 🐍 Python | 🤖 AI | 📊 Database | ☁️ Cloud

Agriculture Icons:  
🌾 Crop | 👨‍🌾 Farmer | 🚜 Tractor | 🌱 Growth | 💧 Water

Business Icons:
💰 Money | 📈 Growth | 🎯 Target | ⚡ Speed | 🏆 Success

Interface Icons:
📱 Mobile | 💻 Laptop | 🌐 Web | 🗣️ Voice | 🔐 Security
```

### **Color Usage Guidelines:**
```
Headings: Agriculture Green (#2E7D32)
Subheadings: Tech Blue (#1976D2)
Highlights: Innovation Orange (#F57C00)  
Success/Positive: Success Green (#388E3C)
Warning/Attention: Amber (#FFA000)
Error/Problems: Red (#D32F2F)
Background: White (#FFFFFF) or Light Gray (#FAFAFA)
Text: Dark Gray (#212121) or Medium Gray (#616161)
```

### **Typography Hierarchy:**
```
Slide Titles: 24pt Montserrat Bold
Section Headers: 20pt Montserrat SemiBold  
Body Text: 16pt Open Sans Regular
Captions: 14pt Open Sans Regular
Small Text: 12pt Open Sans Regular
Code/Technical: 14pt Consolas Regular
```

### **Layout Principles:**
```
✅ Minimum 2cm margins on all sides
✅ Consistent spacing (0.5cm, 1cm, 2cm increments)  
✅ Left-align text blocks for readability
✅ Center-align titles and key statements
✅ Use white space effectively (60% content, 40% space)
✅ Maximum 7 bullet points per slide
✅ Limit text to 6 lines per bullet point
✅ Use high-contrast colors for accessibility
```

## 🖼️ **IMAGE GUIDELINES**

### **Required Images:**
```
Hero Images:
- Indian farmer using smartphone (high-res, authentic)
- Modern agriculture equipment in Indian field
- Diverse crops (wheat, rice, cotton, etc.)
- Rural technology adoption scenes

Screenshots:
- Application interface (clean, high-res)
- Mobile responsive design
- Results dashboard with real data
- Multi-language interface examples

Team Photos:
- Professional headshots (consistent lighting)
- Team group photo (outdoor, casual-professional)
- Working session photos (coding, discussing)
```

### **Image Specifications:**
```
Resolution: Minimum 1920x1080 for full slides
Format: PNG for screenshots, JPG for photos
Quality: High (minimal compression)
Style: Consistent color grading
Backgrounds: Remove or blur distracting elements
Text Overlay: Ensure good contrast and readability
```

## 📤 **EXPORT & DELIVERY SPECIFICATIONS**

### **File Formats:**
```
Primary: PowerPoint (.pptx) - for presentation
Backup: PDF (.pdf) - for compatibility  
Demo: Separate file with embedded videos/GIFs
Handout: Summary version (6 slides per page)
```

### **Technical Settings:**
```
Slide Size: 16:9 widescreen (standard)
Resolution: 1920x1080 pixels
Fonts: Embed all fonts in presentation
Media: Compress images to 150 DPI (balance quality/size)
Transitions: Simple fade (3-second duration max)
Animations: Minimal, professional only
```

This comprehensive visual guide ensures your SIH presentation looks professional, engaging, and technically impressive! 🎨🏆
